# Walmart Sales Analysis

## Overview
Performed end-to-end sales analysis on Walmart data using SQL to identify trends, seasonal patterns, and store-level performance insights.

## Dataset
- 45 Walmart stores
- Weekly sales data (2010–2012)
- Features: sales, holidays, temperature, fuel price, CPI, unemployment

## Tools Used
- PostgreSQL
- SQL

## Key Insights
- Sales peak during November–December, with holiday weeks showing up to 60–70% higher revenue
- Holiday impact varies across stores, with some showing ~19% growth while others decline
- Smaller stores are more sensitive to seasonal demand, while larger stores dominate total revenue

## Business Recommendations
- Focus inventory and promotions on high-impact stores during holidays
- Prepare supply chain for peak demand periods
- Investigate underperforming stores during holidays
